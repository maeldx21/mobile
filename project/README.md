# Audoção - App de Adoção de Pets

App feito em React Native pra facilitar a adoção de pets

## Como rodar

Instalar dependencias:
```
npm install
```

Rodar o projeto:
```
npx expo start
```

Depois é só escanear o QR code com o Expo Go no celular

## Telas

- Home - tela inicial
- Adotar - lista os pets
- Publicar - adiciona novo pet

## Tecnologias usadas

React Native, Expo, AsyncStorage pra salvar os dados localmente
